Welcome

This e-portfolio showcases my learning journey across the MSc Cyber Security programme. Each module folder includes artefacts, reflections, and references that demonstrate knowledge growth, collaboration, and professional development.

Modules

  • Security & Risk Management

    Risk identification, analysis, mitigation, and reflection (ISO 27005, NIST SP 800-30).

  • Launching into Cyber Security

    Foundations of cybersecurity concepts, professional ethics, and digital literacy.

  • Network Security

    Secure network design, protocols, and vulnerability analysis in enterprise systems.

  • Secure Software Development

    Secure coding, STRIDE/PASTA threat modelling, and vulnerability management.

  • Secure System Architecture

    Formal modelling vs. automated testing and architectural design for resilience.

  • The Human Factor

    Human behaviour, insider threat mitigation, and social engineering defences.

  • Research Methods & Professional Practice

    Research design, ethics, and reflective professional practice in cybersecurity.

About

This site is hosted via GitHub Pages and serves as my personal learning record. Each module folder documents both technical and reflective artefacts, peer/tutor feedback, and references in Harvard style.
